# OpenSteelQR

Scan QR → Locate Member → View in 3D

## About

OpenSteelQR is an open-source BIM project that connects QR codes with
Tekla Structures and Trimble Connect to locate steel members in a 3D model.

## Vision

To help site engineers quickly identify and visualize steel members
using QR technology and BIM workflows.

## Technology

- Tekla Structures 2024
- Trimble Connect API
- ASP.NET Core
- Web Viewer
- QR Code Technology
